# AI测试用例生成工具

这是一个基于AI的智能测试用例生成工具，可以帮助测试工程师快速生成专业的测试用例。

## 功能特点

- 🤖 基于OpenAI GPT模型智能生成测试用例
- 📁 支持多种文档格式上传（txt, doc, docx）
- ✍️ 支持直接文本输入，无需上传文件
- 📊 自动生成结构化的测试用例表格
- 📤 支持导出为Excel格式
- 💻 现代化Web界面，操作简单直观
- 🔁 双模式输入：文件上传 + 文本粘贴

## 使用方法

### 1. 启动服务
```bash
python3 main.py
```

### 2. 访问应用
打开浏览器访问：http://localhost:8000

### 3. 使用流程
1. **输入需求内容**：
   - 方式一：上传需求文档（支持txt, doc, docx格式）
   - 方式二：直接粘贴需求文本内容
2. **配置API密钥**：输入您的OpenAI API密钥
3. **生成测试用例**：点击生成按钮，AI将自动生成测试用例
4. **导出结果**：将生成的测试用例导出为Excel文件

## 配置说明

### OpenAI API密钥
您需要在OpenAI官网注册账号并获取API密钥：
1. 访问 https://platform.openai.com/
2. 注册或登录账号
3. 在API Keys页面创建新的密钥
4. 在工具中输入该密钥

### 环境变量（可选）
您也可以通过环境变量设置API密钥：
```bash
export OPENAI_API_KEY="your-api-key-here"
```

## 测试用例字段说明

生成的测试用例包含以下字段：
- **功能模块**：测试的功能模块名称
- **用例编号**：唯一的测试用例ID
- **用例名称**：测试用例的描述性名称
- **前置条件**：执行测试前需要满足的条件
- **测试步骤**：详细的测试执行步骤
- **预期结果**：期望的测试结果
- **优先级**：测试用例的重要程度（高/中/低）
- **用例类型**：测试类型（功能测试/接口测试/性能测试/安全测试）

## 注意事项

## 常见问题

**Q: 为什么需要OpenAI API密钥？**
A: 工具使用OpenAI的GPT模型来理解和分析需求文档，并生成专业的测试用例。

**Q: 生成的测试用例质量如何？**
A: 质量取决于需求文档的详细程度和清晰度。建议提供结构化、详细的需求说明。

**Q: 支持哪些类型的测试用例？**
A: 目前主要支持功能测试用例，后续版本会增加接口测试、性能测试等类型。

**Q: 如何提高生成效果？**
A: 提供更详细、结构化的需求文档，明确功能点和业务逻辑。
